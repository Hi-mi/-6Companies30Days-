Q1 . Minimum size subarray sum(Goldman sachs)
 
 solution- My approach is using two pointers and storing min length of subarray 
 My code - int n=nums.length,l=0,r=0,min=Integer.MAX_VALUE,sum=0;
        while(r<n){
          sum+=nums[r++];
            while(sum>=target){
                min=Math.min(min,r-l);
                sum-=nums[l++];
            }
        }
        return min==Integer.MAX_VALUE?0:min;
    }
}
